Nutzungsbedingungen

Speicherung von Daten

Beim Hinzufügen deines Discord-User zur Halterkarte stimmst Du zu, dass folgende Daten zu deinem Benutzer in der Datenbank abgespeichert werden:

  • Discord User ID
  • Discord Username
  • GPS-Koordinaten des Wohnortes
  • Name des Wohnorts bzw. Adresse

Auffindbarkeit

Du bist außerdem damit einverstanden, dass dein Benutzername auf einer Karte dargestellt wird. Des Weiteren bist Du damit einverstanden, dass dein Benutzername über eine Umkreisabfrage auffindbar ist.
